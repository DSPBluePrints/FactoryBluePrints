## 简介

这是游戏 [**戴森球计划**](https://store.steampowered.com/app/1366540/_/) 中的蓝图仓库，用于储存与分享来自社区的工厂蓝图。

最早一批蓝图主要由 **小马蓝图群** 成员与 [**CIDT 设科院**](https://dspcidt.cn/) 成员贡献。为了对来自社区的庞大数量的蓝图进行分类、管理、储存与分享，我们打算将蓝图上传到 GitHub 作为交流群文件的备份，并且将来可能进一步建设成为蓝图网。

## 蓝图仓库使用方法

> 蓝图仓库已集成 Git 与自动更新脚本，无需任何基础也可以正常使用。

### 第一次下载

1. 在电脑上打开 GitHub 上的蓝图仓库主页：[https://github.com/DSPBluePrints/FactoryBluePrints](https://github.com/DSPBluePrints/FactoryBluePrints)
2. 从此页面 → [**Releases**](https://github.com/DSPBluePrints/FactoryBluePrints/releases) ← 下载最新的蓝图包，文件名应该叫 `FactoryBluePrints_X.Y.Z.rar`。请 **不要** 从绿色的 **Code** 处下载 **Source code**，那样文件不！完！整！
3. 解压后，像其他普通的蓝图包一样，连同文件夹一起放入游戏的默认蓝图文件夹。放进去以后应该是这样：`C:\Users\%USERNAME%\Documents\Dyson Sphere Program\Blueprint\FactoryBluePrints\README.md`。
4. **第一次使用必须先更新，否则此时没有任何蓝图。**

### 更新

1. 只需双击即可：`update.bat`

## 常见问题

> Q：为什么我打不开蓝图仓库主页 / 下载特别慢 / 更新特别慢？
> A：通常是网络问题。蓝图仓库被托管在 GitHub 上，可以通过在 [Watt Toolkit](https://steampp.net)（原名 Steam++）中开启 GitHub 加速服务解决。~~或者魔法？~~

> Q：警告：您似乎安装到了错误的路径，这可能导致文件权限异常  
> A：通常是用户解压后，全选文件放入 `Blueprint`，而不是连同文件夹一起放入 `Blueprint`。蓝图仓库不依赖绝对路径，但这并不代表在任意路径下 Git 都有读写文件的权限；而在部分电脑上，Git 无权访问 `Blueprint` 根目录。
